# Decree 130/2018/ND-CP Reference

**Title**: Decree No. 130/2018/ND-CP detailing the implementation of the Law on E-Transactions regarding digital signatures and digital signature certification services.

**Official Download Link**: [Government Portal](https://datafiles.chinhphu.vn/cpp/files/vbpq/2018/09/130-2018-nd-cp.signed.pdf)

**Status**: The direct download via automated tools was restricted. Please use the link above to view the official signed PDF.

**Key Regulations relevanta to this project**:
1.  **Digital Signature Validity**: Must be created by a secure key under sole control.
2.  **CA Requirements**: Public CAs must be licensed by MIC.
3.  **Government CA**: The Government Root CA certifies the specialized CAs.
